Programming Languages

IUseArchBtw (GitHub)

IUseArchBtw is an esoteric programming language based on a BrainF. There are only a few commands that the language recognizes. i, use, arch, btw, notarch, not.

Here is an example program that adds two inputed numbers together. 

                            
    1 i 
    2 arch i
    3 [
    4     notarch use
    5     arch not
    6 ]
    7 use use use use
    8 use use use use  
    9 [
    10     notarch use use use use use use
    11     arch not
    12 ]
    13 notarch btw
And here is the output 
                            
    1 $ iusearchbtw ./tests/add.archbtw
    2 7 
    3 2
    4 9
